Projects

Algorithms, AI and Web App, May 2017 - Present

Music Museum

Based by playlists, songs, authors it finds different dependencies and music similarity between the data and gives you visualized result, so that you can analyze the data. You can use this free Spotify account for testing (username: museum-test, password: 12345678).

Additionally, you can upload your own audio file to analyze its melody. Artificial Intelligence will predict the genre of the uploaded song and show you how the melody is changing over time. The model was created during Classifying an audio signal using Neural Networks and Manifold Learning project and then published to Music museum project.

Spotify API is used to get info about songs and playlists. I have used Push–relabel maximum flow and Minimum cut algorithms to analyze the given data.

https://github.com/olderor/museum

Machine Learning: Neural Networks and Manifold Learning, Jan 2019 - Mar 2019

Classifying an audio signal using Neural Networks and Manifold Learning

The main idea is to be able to classify the audio signal into some predefined groups. We can use a genre as a group that describes some songs that have something similar with each other. You can set some groups (like genres) to some audio signals and teach the model to find similarity between those signals. So that you will be able to predict the group for a new audio signal.

https://github.com/olderor/data-structures

Data Structures & Mobile App, Nov 2016 - Dec 2016
